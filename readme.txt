著作権ジェネレータープラグイン（Copyrights generator plugin）

== Description ==

It is a plug-in that automatically creates copyright notation on the website.
When the New Year comes, you can change the copyright renewal year automatically and display it.

The set copyright notation is pasted to the site theme footer etc. with a short code.

Webサイトに著作権表記を自動で作成してくれるプラグインです。
新年が来たら自動で著作権更新年を変更して表示できます。

設定した著作権表記はショートコードでサイトテーマフッターなどに貼り付けるだけです。


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
